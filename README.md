## Install

```bash
git clone <your-repo-url>
cd extension
npm install
npm run dev
```

## Load in Chrome
- Open `chrome://extensions`
- Enable **Developer mode**
- Click **Load unpacked**
- Select the `dist` folder

## Add API Key
- Click the **Extensions** icon in Chrome
- Find **Agentic Browser**
- Click the **Options** page
- Enter your **API key**
